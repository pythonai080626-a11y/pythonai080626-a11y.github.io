Topic 04

Conditionals — if / else 🔀

The moment your programs start making decisions. This is the Python translation of the flowchart's diamond: run some code only when a condition is true with if, else, and elif — and learn why indentation is everything.

age.py
age = int(input()) if age >= 18: print('Beer') else: print('Coke')
1

if — Do Something Only When It's True

An if statement checks a condition. If the condition is True, the indented lines underneath run. If it's False, Python skips them. Notice the colon : at the end of the if line.

age = int(input('age?')) if age >= 18: print('You are an adult') print('done') # runs no matter what
age?20 You are an adult done
💡 The diamond in a flowchart = an if. The YES arrow is the indented block.
2

else — The Other Path

else catches every case where the if condition was False. Exactly one of the two blocks always runs — never both, never neither.

age = int(input('age?')) if age >= 18: print('You get a Beer!') else: print('You get a Coke!')
age?15 You get a Coke!
3

Indentation Is the Block ⚠️

Most languages use { } braces to group lines. Python uses spaces. The lines pushed in by 4 spaces (the highlighted part) belong to the if. The moment a line goes back to the left margin, it is outside the if.

if age >= 18: print('line A') # inside — only if True print('line B') # inside — only if True print('line C') # outside — always runs
✅ Press Tab once to indent, or type 4 spaces. PyCharm indents automatically after a line ending in :.
⚠️ Mixing tabs and spaces, or the wrong indent, gives an IndentationError. Keep it consistent.
4

Conditions — The Comparison Operators

A condition is any expression that is either True or False:

==  equal to
!=  not equal
>  greater than
<  less than
>=  greater or equal
<=  less or equal
⚡ One equals sign = assigns a value. Two == compares. Beginners mix these up constantly!
5

Nested if — A Decision Inside a Decision

Sometimes one check isn't enough. You can put an if inside an else. Each level of nesting is pushed in by 4 more spaces. Example: print the bigger of two numbers, or Equal.

a = int(input('a?')) b = int(input('b?')) if a > b: print(a) else: if a == b: print('Equal') else: print(b)
a?3 b?7 7
😅 Notice how the code marches further and further to the right. With three or four choices this "staircase" gets hard to read — that's exactly the problem elif solves.
6

elif — Else, But Check Again

elif is short for "else if". It flattens the nested staircase into one clean ladder. Python checks each condition from top to bottom and runs the first one that is True — then skips the rest.

grade = int(input('grade?')) if grade >= 90: print('A') elif grade >= 80: print('B') elif grade >= 70: print('C') else: print('Fail')
grade?85 B
💡 You can have as many elif branches as you like. else is optional and always goes last — it catches everything the conditions above missed.

Nested if vs elif — Same Logic, Cleaner Code

These two programs behave identically. The elif version is easier to read and to extend:

😰 Nested — the staircase
if x == 1: print('one') else: if x == 2: print('two') else: print('other')
😎 elif — the ladder
if x == 1: print('one') elif x == 2: print('two') else: print('other')

Quick Reference

Every block needs
a : colon at the end of the line, then 4 spaces of indentation on the lines below.
Order matters
if → optional elif(s) → optional else last.
= vs ==
= stores a value · == compares two values.
Combine conditions
use and (both true) and or (at least one true).
