Topic 03

Python — First Steps 🐍

Time to turn flowcharts into real programs! This tutorial walks you through creating your first project in PyCharm, running code with a single shortcut, and the two functions you'll use in every program: print and input.

hello.py
# my first program print('What is your name?') my_name = input() print("Hello", my_name)
1

Create a New Project in PyCharm

A project is the folder where all your Python files live. You create it once, then add as many files as you want.

  1. Open PyCharm and click File → New Project (or New Project on the welcome screen).
  2. Give the project a name — for example python_class.
  3. Leave the interpreter settings as they are (PyCharm prepares Python for you).
  4. Click Create.
💡 PyCharm may create a main.py file with sample code inside — you can delete its content or ignore it. We will write our own files.
2

Add a Python File

Each program goes in its own .py file.

  1. In the project panel on the left, right-click the project folder.
  2. Choose New → Python File.
  3. Type a name — for example hello — and press Enter. PyCharm adds the .py for you.
3

Run It — Shift + F10

Type one line in your new file, then run it:

print("Hello Itay!")

Three ways to run — pick your favorite:

  • Press Shift + F10 — runs the last file you ran.
  • Click the green ▶ button in the top toolbar (or in the gutter next to the code).
  • Right-click inside the file → Run 'hello'.

The result appears in the Run window at the bottom:

Hello Itay! Process finished with exit code 0
💡 "exit code 0" means the program finished with no errors. You will learn to love that line.
4

print() — Show Something on Screen

print is the Python version of the flowchart's output shape. Each print() writes one line.

print("Hello Itay!") # text goes in quotes print(1) # numbers need no quotes print(2) print(3)
Hello Itay! 1 2 3
5

Comments — Notes Python Ignores

A line starting with # is a comment — Python skips it completely. Use comments to explain your code, or to mark the START and STOP of your flowchart.

# START print("only this line runs") # STOP
⚡ Select one or more lines and press Ctrl + / — PyCharm comments or uncomments them all at once.
6

input() — Get an Answer from the User

input() is the flowchart's input shape. It pauses the program, waits for the user to type a line and press Enter, and hands you whatever they typed — which you store in a variable.

# START print('What is your name?') my_name = input() print("Hello", my_name) # STOP
What is your name? Itay Hello Itay
💡 In print("Hello", my_name) the comma prints both values together, with a space between them.

Flowchart → Python Cheat Sheet

Every shape you learned in Flowcharts Basics has a Python translation:

START / STOP oval
# START and # STOP comments
Output shape
print(...)
Input shape
