draw.io

Creating Diagrams

We used draw.io to create visual diagrams.

Main steps
  1. Open draw.io.
  2. Create a new diagram.
  3. Add shapes and arrows.
  4. Save or export the diagram file.
draw.io helps us explain an idea visually before writing or sharing files.
GitHub

Creating a GitHub User

We created a GitHub user so each student can save and share work online.

Main steps
  1. Open github.com.
  2. Choose Sign up.
  3. Enter an email, password, and username.
  4. Confirm the account if GitHub asks for verification.
The username is public, so choose a name that is simple and easy to recognize.
Repository

Creating a Repository

A repository is a folder on GitHub where project files are stored.

Main steps
  1. Click the New repository button.
  2. Write a repository name.
  3. Choose whether the repository is public or private.
  4. Create the repository.
A clear repository name makes it easier to find the correct project later.
Files

Uploading Files to GitHub

We uploaded files into a GitHub repository so the work is saved online.

Main steps
  1. Open the repository.
  2. Click Add file.
  3. Choose Upload files.
  4. Drag files into the upload area or select them from the computer.
  5. Click Commit changes.
Committing changes saves the uploaded files inside the repository.
