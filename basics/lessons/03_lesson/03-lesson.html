If / Else

An if block runs code only when a condition is True.
An else block runs when the condition is False.

Every if must have a condition. The else is optional — use it when you need a fallback.

If / Elif / Else

Use elif (else-if) to check multiple conditions one after another. Python checks each in order and stops at the first one that is True.

Python checks each condition top-to-bottom and runs only the first block that matches. You can chain as many elif as you need.

Modulus (%)

The % (modulus) operator returns the remainder after division.

ExpressionResultWhy
10 % 3110 ÷ 3 = 3 remainder 1
8 % 208 ÷ 2 = 4 remainder 0
7 % 217 ÷ 2 = 3 remainder 1
15 % 5015 ÷ 5 = 3 remainder 0
11 % 4311 ÷ 4 = 2 remainder 3
A number is even when number % 2 == 0. It is odd when number % 2 == 1.
Integer division (//)

The // operator divides and drops the decimal (floor division).

ExpressionResultNote
10 // 333 full groups of 3 in 10
7 // 23not 3.5 — decimal part dropped
15 // 53exact, no remainder
