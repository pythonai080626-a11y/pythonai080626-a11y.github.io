Prime Number

A prime number is a number bigger than 1 that can be divided only by 1 and itself. Examples: 2, 3, 5, 7, 11.

The plan:
1. Input a
2. If a < 1, print Enter positive number
3. If a == 1, print Not Prime
4. Start with i = 2
5. While a % i != 0, increase i
6. If i == a, print Prime; otherwise print Not Prime

Python · Program 5

if / else — Beer or Coke?

Our first decision in real code. Read the customer's age; if they are 18 or older serve a beer, otherwise a coke. Either way, say goodbye at the end.

The diamond in the flowchart becomes an if statement. The YES path is the if block, the NO path is the else block. Both paths join again afterwards — that is why goodbye.... always prints.
# START # int() converts the typed text into a number customer_age = int(input("age?")) if customer_age >= 18: print('You get a Beer!') else: print('You get a Coke!') print('goodbye....') # STOP
Output (user types 20)
age?20 You get a Beer! goodbye....
Watch the indentation. The lines under if and else are pushed in with 4 spaces — that is how Python knows which lines belong to each branch. goodbye.... is not indented, so it runs no matter what.
Python · Program 6

if / else if — The Bigger Number

Read two numbers and print the bigger one. If they are equal, print Equal. This needs a second decision inside the first — an if nested inside an else.

# START a = int(input("a?")) b = int(input("b?")) if a > b: print(a) else: if a == b: print("Equal") else: print(b) # STOP
Output (a = 7, b = 3)
a?7 b?3 7
Notice the deeper indentation: the inner if / else lives inside the outer else, so it is pushed in one extra level (8 spaces). Each level of nesting adds 4 more spaces.
Python · Program 7

and — Login Check

Ask for a username and a password. Only log the user in when both are correct. The word and means every condition must be True at the same time.

# START user_name = input('user name?') password = input('password?') if user_name == 'admin' and password == '1234': print('You are logged in') else: print('Fail to login') # STOP
Output (correct login)
user name?admin password?1234 You are logged in
and is strict: if even one side is False, the whole condition is False. Wrong username or wrong password → the else runs and login fails.
Python · Program 8

or — Legal Speed?

A speed is legal if it is slow, fast, or medium. The word or means the condition is True when at least one part matches.

# START speed = input('speed?') if speed == 'slow' or speed == 'fast' or speed == 'medium': print('Accepted') else: print('Illegal speed') # nicer way: # if speed in ['slow', 'fast', 'medium']: ... # STOP
Output (user types fast)
speed?fast Accepted
and vs or: and needs every part True, or needs just one part True. Here any of the three speeds is enough to be accepted.
Python · Program 9

% — Even or Odd (Zugi / E-Zugi)

Read a number and decide if it is even or odd. A number is even (zugi / זוגי) when it divides by 2 with no remainder — that is number % 2 == 0.

# START number = int(input('number?')) if number % 2 == 0: print('Zugi') else: print('E-Zugi') # STOP
Output (user types 4)
number?4 Zugi
% (modulo) gives the remainder of a division. 4 % 2 = 0 → even (Zugi), while 7 % 2 = 1 → odd (E-Zugi). This same trick works for "divisible by 3", "divisible by 10", and so on.
