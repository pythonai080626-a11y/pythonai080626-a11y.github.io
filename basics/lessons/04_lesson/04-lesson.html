The While Loop

A while loop repeats a block of code as long as a condition stays True.
Every time the body finishes, Python checks the condition again. When it becomes False, the loop stops.

How it works — step by step
StepWhat happens
1Check the condition
2If True → run the body, then go back to step 1
3If False → exit the loop, continue the program
3 essential parts of every while loop:
1. Initialise — set the variable before the loop
2. Condition — checked before each repetition
3. Update — change the variable inside the body so the loop eventually stops
Always make sure something inside the loop body changes the condition variable — otherwise the loop runs forever (infinite loop)!

Program 11

Input Until Negative — Sentinel Value

Input a number. While the number is >= 0, print a message and input again. When the user enters a negative number, print "Negative number was entered" and stop.

A sentinel value is a special input that signals "stop the loop". Here the sentinel is any negative number.
The first input happens before the loop so the condition can be checked immediately.
In-class exercise (from diagram):
Input numbers until the user enters 999. Calculate the sum of all numbers entered (not counting 999).
Example: 1, 10, 999 → sum = 11

Sum Until 999 — Accumulator

sum = 0 before the loop is called an accumulator — it collects a running total. Always initialise accumulators to 0 before the loop starts.

Program 14

Multiply Using Addition

Input two positive numbers a and b. Print a × b without using * — use only +.

Multiplication is just repeated addition:
5 × 3 = 5 + 5 + 5 = 15
Start with sum = 0, add a to sum exactly b times.

Program 15

Class Average — Counter Loop

Input the number of students. Then in a loop, input each student's grade and accumulate the total. After the loop, print the average.

i is a counter — it counts how many times the loop body has run.
Pattern: initialise i = 0 before the loop · check i < n · update i += 1 inside the body.
