Diagram 16

Factorial (עצרת)

Factorial of a number is the product of all integers from 1 up to that number:
3! = 1 × 2 × 3 = 6  ·  5! = 1 × 2 × 3 × 4 × 5 = 120

The plan:
1. Input number
2. Set result = 1
3. Set i = 1
4. While i < number — increase i, and each time result = result * i
Trace table (number = 5)
i < number ?i afterresult after
1 < 5 → True21 × 2 = 2
2 < 5 → True32 × 3 = 6
3 < 5 → True46 × 4 = 24
4 < 5 → True524 × 5 = 120
5 < 5 → Falseexit loop → print 120
Notice the trick: we start with result = 1 (not 0!) because we are multiplying — anything multiplied by 0 would stay 0 forever. For products, initialise with 1; for sums, initialise with 0.
Diagram 17

Sum of Digits

Input a number and calculate the sum of its digits.
Example: 123 → 1 + 2 + 3 = 6  ·  456 → 4 + 5 + 6 = 15

The plan:
1. Input number
2. Set sum = 0
3. While number > 0:
   · digit = number % 10 — take the last digit
   · add digit to sum
   · number = number // 10 — chop off the last digit (integer division)
Trace table (number = 456)
number > 0 ?digitsum afternumber after (// 10)
456 > 0 → True456 % 10 = 60 + 6 = 645
45 > 0 → True45 % 10 = 56 + 5 = 114
4 > 0 → True4 % 10 = 411 + 4 = 150
0 > 0 → Falseexit loop → print 15
Two power tools working together:
% 10 gives the last digit (456 % 10 = 6)
// 10 removes the last digit (456 // 10 = 45)
Repeat until the number becomes 0 — you have visited every digit.
Diagram 18

Find Modulus — Without Using %

Input mone (מונה) and mehane (מכנה) and compute mone % mehane — without using the % operator.
Examples: 9 % 4 = 1  ·  11 % 3 = 2  ·  12 % 3 = 0

The idea: keep adding mehane to itself until we reach or pass mone.
1. sum = mehane
2. While sum < mone: sum += mehane
3. If sum == mone → the remainder is 0 (divides evenly)
4. Else we overshot → step one back: sum -= mehane, then the remainder is mone - sum
Trace table (mone = 11, mehane = 3)
sum < mone ?sum after
3 < 11 → True3 + 3 = 6
6 < 11 → True6 + 3 = 9
9 < 11 → True9 + 3 = 12
12 < 11 → Falseexit loop
sum == mone ? 12 == 11 → Falsesum = 12 − 3 = 9 → print 11 − 9 = 2 ✓
And when it divides evenly? (mone = 12, mehane = 3)
StepResult
sum grows: 3 → 6 → 9 → 12loop stops (12 < 12 is False)
sum == mone ? 12 == 12 → Trueprint 0 ✓ (12 % 3 = 0)
mone (מונה) = the number being divided · mehane (מכנה) = the number we divide by.
This is the same trick as lesson 4's "multiply with +" — we rebuild a complex operator from simple ones.
Python · Program 1

🎉 Our First Python Program — START / STOP

Big milestone: today we moved from drawing flowcharts to writing real Python code in PyCharm.
Every flowchart shape has a Python translation — and we begin with the simplest flowchart of all:

A program that starts and immediately stops — it does nothing, and it is 100% valid Python:

# START # STOP
Lines starting with # are comments — Python ignores them completely. They are notes for humans. Here the whole program is just two comments marking where the flowchart starts and stops.

Python · Program 2

Print Hello

Next flowchart: START → output a message → STOP. The output parallelogram becomes the print() function in Python.

# START print("Hello Itay!") # STOP
Output
Hello Itay!
Anatomy of the line: print is a built-in function, the parentheses ( ) call it, and the text in quotes "Hello Itay!" is a string — the value we ask it to display.
Python · Program 3

Print 1, 2, 3 — Instructions Run in Order

Three output boxes, one after another. Python executes the file top to bottom, exactly like following the arrows in the flowchart.

# START print(1) print(2) print(3) # STOP
Output
1 2 3
Each print() writes on a new line. Notice the numbers have no quotes — 1 is a number, while "1" would be a string. Both would display the same here, but they are different types.
Python · Program 4

Input a Name — Talk to the User

Now the full trio: output a question, input an answer into a variable, and output a greeting that uses it. The input shape becomes the input() function.

# START print('What is your name?') my_name = input() print("Hello", my_name) # STOP
Output (user types "Itay")
What is your name? Itay Hello Itay
Flowchart shape → Python translation
Flowchart shapePython
START / STOP oval# START / # STOP comments (beginning and end of the file)
Output parallelogramprint(...)
Input shapevariable = input()
Three new things in one program:
1. input() pauses the program and waits for the user to type a line and press Enter.
2. Whatever they typed is stored in the variable my_name.
3. print("Hello", my_name) — separating values with a comma prints them together, with a space in between.
