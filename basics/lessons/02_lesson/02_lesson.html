Start / Stop

A flowchart shows the steps of a program from beginning to end. Every flowchart must have a clear START and a clear STOP.

Oval: program begins
Oval: program ends
Rule: nothing comes before START and nothing comes after STOP. The arrows show the order of the program.

Input / Output

Input means the user gives data to the program. In Python, this is usually input().

Output means the program shows data to the user. In Python, this is usually print().

Parallelogram: data goes in
Parallelogram: data goes out
Python examples
num = int(input("Enter a number: ")) # INPUT print(num) # OUTPUT

Operation

An operation is a step where the computer does work: calculation, assignment, or changing a value. It is drawn as a rectangle.

Python examples
Flowchart operationPython code
square = num * numsquare = num * num
total = price * quantitytotal = price * quantity
cars = children / 5cars = children // 5
Use a rectangle when the program calculates something but does not ask the user and does not print yet.

Print two numbers

Print the square

InputExpected outputWhy

Class trip cars

There is a class trip. Each car has 5 seats. Input how many children are in the class, then print how many cars need to be booked.

The number of children is guaranteed to be divisible by 5, so no rounding is needed.

ChildrenCars neededWhy
10210 / 5 = 2
15315 / 5 = 3
20420 / 5 = 4
25525 / 5 = 5
515 / 5 = 1
Your flowchart should use: START, one INPUT box, one OPERATION box for cars = children / 5, one OUTPUT box, STOP.
